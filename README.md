# numeric_recognition


How to use


~環境構築~  
仮想環境の構築  
python3 -m venv count_num  
仮想環境のアクティベート  
source ./count_num/bin/activate  
依存ライブラリのインストール  
pip install -r requirements.txt  


~実行~  
dataフォルダの中にmove.mp4という名前のファイルを追加する  
python3 count_mov.py  
tableフォルダの中にカウントされたデータが出力される  
なお上書きされたるため、実行するたびにバックアップを取ること
